## Цель работы

Создание персонального сайта с использованием MkDocs и языка разметки Markdown.

---

## Ссылка на сайт

https://voroshninaeva111-lgtm.github.io/my-personal-site/

---

## Этап 1. Подготовка и установка

1. Проверена установленная версия Python.
2. Установлен MkDocs с помощью команды:
   `pip install mkdocs`
3. Установлена тема Material:
   `pip install mkdocs-material`
4. Проверена установка:
   `mkdocs --version`

---

## Этап 2. Создание проекта

1. Создана папка проекта `my-personal-site`.
2. Выполнен переход в папку проекта.
3. Инициализирован проект командой:
   `mkdocs new .`
4. Изучена структура проекта:

   * `mkdocs.yml`
   * папка `docs/`

---

## Этап 3. Настройка конфигурации

1. Отредактирован файл `mkdocs.yml`.
2. Настроено название сайта.
3. Подключена тема Material.
4. Настроена навигация по страницам.

---

## Этап 4. Создание контента

1. Создана главная страница.
2. Добавлены дополнительные страницы:

   * Обо мне
   * Проекты
   * Контакты
3. Добавлено изображение на главной.
4. Использованы элементы Markdown:

   * заголовки
   * списки
   * ссылки
   * изображения

---

## Этап 5. Публикация сайта

1. Создан репозиторий на GitHub.
2. Локальный проект связан с удалённым репозиторием.
3. Выполнена публикация с помощью команды:
   `mkdocs gh-deploy`
4. Сайт опубликован через GitHub Pages.

---

## Итог

В ходе выполнения работы был создан персональный сайт с использованием MkDocs.
Сайт успешно опубликован и доступен по публичной ссылке.
